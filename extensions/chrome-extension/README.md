# Vive Coders Security - Chrome Extension

Scan GitHub PRs and deployment pages for security issues.

## Installation

1. Open Chrome and go to `chrome://extensions/`
2. Enable "Developer mode"
3. Click "Load unpacked"
4. Select the `chrome-extension` directory

## Usage

- Navigate to a GitHub PR page
- The extension will automatically scan the PR
- Security findings will be displayed at the top of the page
- Click the extension icon to manually trigger a scan

## Features

- Automatic PR scanning
- Security findings display
- Integration with Vive Coders Security API
